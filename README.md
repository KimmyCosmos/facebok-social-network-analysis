# Facebook Social Network Analysis

## Overview
This project analyzes structural properties of a large-scale social network using the Facebook Combined Network dataset from the Stanford SNAP repository. The analysis explores network connectivity, degree distributions, centrality measures, community detection, and link prediction techniques.

## Methods
- Network density analysis
- Degree distribution and power-law analysis
- Centrality metrics (degree, betweenness, closeness)
- Community detection (Greedy Modularity and Girvan–Newman)
- Bridge node analysis
- Link prediction using Jaccard and Adamic–Adar indices

## Data
Facebook Combined Network  
Stanford SNAP Dataset  
https://snap.stanford.edu/data/facebook_combined.txt.gz

## Tools
Python, NetworkX, NumPy, Matplotlib
